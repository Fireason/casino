# 🎰🎰🎰🎰 Weather Casino 🎰🎰🎰

---

## 📌 Описание
Рандомно генерируется координата (и ближайший к ней город, возможно), надо угадать погоду или наличие дождя 

Пользователь делает ставку:

* будет ли дождь ☔
* будет ли температура > X🌡️

После этого система проверяет реальную погоду через API и определяет:

* выиграл пользователь или проиграл

---

## 🏗️ Архитектура проекта

* Django (backend)
* SQLite (база данных)
* Weather API (Open-Meteo)
* HTML шаблоны (frontend)
